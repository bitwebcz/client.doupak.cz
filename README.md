# doupak - vue 3 learning app

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Runs unit tests
```
yarn test
```

### Deploys to gh-pages
```
yarn deploy
```
